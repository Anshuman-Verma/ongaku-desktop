# ongaku-desktop

> Electron based desktop port of [ongaku](https://github.com/Anshuman-Verma/ongaku).

Play anime music with ease on your desktop.

## Dev

```
$ git clone --recursive https://github.com/aviaryan/emojiui.git
$ cd ongaku-desktop
$ npm install
```

## Run

```
$ npm start
```

If installed globally

```
$ ongaku
```

### Build

```
$ npm run build
```
